# X (Twitter) Clone

A responsive **X (Twitter) clone UI** built using **HTML and Tailwind CSS**.  
This project replicates the basic layout of the X platform including the sidebar navigation, main feed section, trending topics, and user suggestions.

##  Features
- Responsive layout
- Sidebar navigation similar to X
- Feed section design
- Trending topics section
- Modern UI built with Tailwind CSS

##  Technologies Used
- HTML
- Tailwind CSS
